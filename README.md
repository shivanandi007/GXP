# GXP — Grow Perform Landing Page

A zero-cost, static landing page for the GXP fitness app/waitlist.

## Files

- `index.html` — main website
- `1000511459.jpg` — used as hero + football athlete card
- `1000511446.jpg` — used as male aesthetic/gym card
- `1000511447.jpg` — used as lean/Virat-style card
- `1000511448.jpg` — used as pull-up/functional strength card

> If any image appears in the wrong place, rename the files or update the image URLs in `index.html` under the `.ref-img.one`, `.ref-img.two`, `.ref-img.three`, `.ref-img.four`, and `.hero::before` CSS rules.

## Deploy Free on GitHub Pages

1. Create a GitHub repo named `gxp-landing`
2. Upload `index.html`, `README.md`, and the 4 JPG images to the repo root
3. Go to **Settings → Pages**
4. Source: **Deploy from branch**
5. Branch: `main`, Folder: `/root`
6. Save
7. Your site will be live at:

```txt
https://YOUR_USERNAME.github.io/gxp-landing/
```
